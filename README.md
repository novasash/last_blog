# Last blog

### Objective
To refresh memories on Ruby on Rails and to learn Turbo 8 features for live updates without page refresh.

### Prerequisites 
- Ruby `ruby 3.4.1`
- Rails `Rails 8.0.1`
- Rich text `action_text`
- CSS `TailwindCSS 4.0`

### Project setup 
- Run `bundle install` to install dependencies in `Gemfile`
- To start a server use `bin/dev` command

### Database creation
Run the following commands to set up a project database
1. `rails db:create`
2. `rails db:migrate`
3. `rails db:seed`

You should be all set.
